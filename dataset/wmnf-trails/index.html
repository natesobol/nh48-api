WMNF Trails Dataset

Download normalized trail data for the White Mountain National Forest, including New Hampshire core coverage, Maine extensions, connector graphs, and supporting peak references.

The dataset is published as static files so search engines and downstream apps can consume the metadata without relying on client-side rendering.

What’s inside

This collection focuses on open, machine-readable trail data for the White Mountain National Forest.

  • Segment-level details: trail names, length, region grouping, and a normalized version for network analysis.
  • Coverage for the New Hampshire core, Pliney region, and Maine segments to ensure continuity across the forest boundary.
  • Trailhead and peak references to align the network with nearby parking, access points, and the 2,000-foot peak list.
  • JSON exports
  • Spreadsheet & text extracts
  • Normalized graph
  • WMNF focus
  • No authentication required

How to use this dataset

  • Render WMNF hiking routes on static or interactive maps using the normalized network.
  • Analyze mileage, coverage, or segment density for trip planning or routing experiments.
  • Join the trail data with peak references to support guidebook-style summaries or offline GPX exports.
