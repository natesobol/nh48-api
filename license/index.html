NH48 API Open Data

Creative Commons Attribution 4.0

All NH48 API datasets are published under the Creative Commons Attribution 4.0 International License (CC BY 4.0). You are free to share and adapt the data for any purpose with proper attribution to Nathan Sobol and NH48 API.

License overview

What CC BY 4.0 allows

  • Share: copy, distribute, and transmit the datasets in any format.
  • Adapt: remix, transform, and build upon the data for any purpose, even commercially.
  • Attribute: credit “NH48 API / Nathan Sobol” with a link to this license page or the official CC BY 4.0 notice.
View CC BY 4.0 Browse datasets

How to cite NH48 API data

Reference the license in your metadata or citations using either link below:

  • https://creativecommons.org/licenses/by/4.0/
  • https://nh48.info/license

Example attribution: “Data from NH48 API (Nathan Sobol), licensed under CC BY 4.0.”

Why this matters

  • Clear licensing improves dataset discovery in Google Dataset Search.
  • Trusted attribution signals encourage reuse by researchers and mapping apps.
  • Consistent CC BY 4.0 coverage keeps downstream mirrors and caches compliant.
