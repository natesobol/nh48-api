About Nathan Sobol

Landscape photographer, route researcher, and founder of NH48pics.

I built NH48.info as a destination for hikers and researchers who want more than a raw API feed: complete peak guides, route context, first-hand planning notes, and photo-backed metadata from across New Hampshire's 4,000-footers.

Experience Timeline

I have documented all 48 New Hampshire 4,000-foot peaks through multi-season hiking and field photography.

Each guide is reviewed against trail conditions, weather exposure, and route decisions encountered on the mountain.

NH48pics combines guide writing, structured data publishing, and licensing-safe image metadata for each location.

Methodology

Peak pages are built from source data, route records, geospatial references, and on-trail observations.

Photo metadata is aligned to visible page images so schema reflects real media shown to users and crawlers.

Updates are tracked through reproducible scripts and audits in CI before worker deployment.

Project Scope

NH48.info publishes peak guides, range context, trail references, photo galleries, and developer-ready datasets.

The API remains available, but the site is organized first for hikers and search users exploring White Mountains content.

Questions, corrections, or licensing requests can be sent through the submit edit form.
